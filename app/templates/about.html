About

This service lets you analyze, visualize and share YSQL Query Plans. It will compute and highlight the most important information to make them easier to understand.

It is brought to you by Yugabyte Support Team.

Usage tips

Format

To get the most out of it, we recommend generating a query plan using the following line: EXPLAIN (ANALYZE, DIST, COSTS, VERBOSE, BUFFERS, FORMAT JSON). It is recommend submitting a (decently formatted) SQL query that generated the plan.

PEV2 can also parse text although the JSON format usually has better support.

POST

You can send plans using HTTP POST to {{ url_for('new', _external=True) }}.
The request parameters to send are title, plan and optionaly query.
Here's an example using a HTML form: 

<form action="{{ url_for('new', _external=True) }}" method="POST">
  <input type="text" name="title" value="A title for the plan"/>
  <textarea name="plan" rows="10" cols="50">
  Seq Scan on tenk1  (cost=0.00..333.00 rows=10000 width=148)
  </textarea>
  <textarea name="query" cols="50"></textarea>
  <button type="submit">Submit</button>
</form>
Here's an example using cURL:
$ curl -Ls -w %{url_effective} -d '{"plan":"Seq Scan on tenk1 (cost=0.00..333.00 rows=10000 width=148)"}' -H "Content-Type: application/json" -X POST {{ url_for('new', _external=True) }} -o /dev/null

Directly from psql

You can send your plans directly from psql using the plan-exporter utility.

Help us improve this service

This application uses PEV2 JS library to analyze and display the plan.

Data retention policy

The plans you send are stored in the internal postgresql database deployed GCP instance. This allows you to easily share a link to anyone.

It is recommended not to send any critical or sensitive information.

Plans are meant to be stored permanently (with no waranty) unless you delete them yourself.

You can delete the saved plans using the list shown in the home page. Make sure you're using the same browser.
